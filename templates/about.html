ℹ️ About This Project

Job Skill Portal — A comprehensive full-stack AI project with BST data structure

📋 Project Overview

The Job Skill Portal is a full-stack web application that combines Machine Learning with fundamental Data Structures to create an intelligent job matching system. The system is trained on 23,427 real job postings across 5 industry categories, using a Random Forest classifier to predict the most suitable job category based on user skills.

At its core, the application uses a Binary Search Tree (BST) to index and organize all extracted skills from the dataset. This allows efficient O(log n) search operations and provides a visually stunning interactive tree visualization powered by D3.js.

🌳 Binary Search Tree (BST) — Algorithm Details

The BST is the core data structure of this project, used for skill indexing and lookup:

Insert — O(log n)

Skills are inserted alphabetically. If a skill already exists, the job ID is added to its list.

Search — O(log n)

Efficiently find any skill and retrieve all associated jobs. Search path is animated in the visualizer.

Delete — O(log n)

Handles all three cases: leaf node, one child, and two children (using inorder successor).

Traversals

Inorder (sorted), Preorder (root-first), and Postorder (leaves-first) — all with animated visualization.

🤖 Random Forest Classifier

The prediction engine uses a Random Forest ensemble of 200 decision trees trained on TF-IDF vectorized skill features. The model achieves high accuracy by:

  • Cleaning and normalizing 23,427 skill set strings
  • TF-IDF vectorization with 5,000 features and bigrams
  • Balanced class weights for fair predictions across all 5 categories
  • 200 estimators for robust ensemble predictions
  • Probability-based ranking of all categories for each prediction
